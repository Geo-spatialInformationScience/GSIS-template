# LaTeX Template for Geo-spatial Information Science (GSIS) Journal

[![LaTeX](https://img.shields.io/badge/LaTeX-project-008080.svg?style=flat&logo=latex)](https://www.latex-project.org/)
[![Journal](https://img.shields.io/badge/Journal-GSIS-blue.svg)](https://www.tandfonline.com/journals/tgsi20)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This repository provides an official LaTeX template tailored to meet the formatting and submission guidelines of the **Geo-spatial Information Science (GSIS)** journal (published by Taylor & Francis). It includes pre-configured document classes, citation styles, and structural layouts suitable for remote sensing, photogrammetry, GIS, and spatial data processing manuscripts.

---

## 📂 Project Structure

```text
├── manuscript.tex           # Main LaTeX source file (write your paper here)
├── gsis.cls                 # GSIS document class file (layout, fonts, margins)
├── tfcad.bst                # Format for your references
├── ref.bib                  # BibTeX database file for your citations
├── README.md                # Project documentation
└── figures/                 # Directory containing all illustration/chart files
    └── figure1.png
